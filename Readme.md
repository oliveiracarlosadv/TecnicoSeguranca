## Como usar
1. Abra o arquivo **Index.html** em um navegador moderno.  
2. Ao abrir, a interface fará a pergunta obrigatória: **"Que tipo de SKILL quer criar?"**. Responda com o foco desejado (ex.: "Investigação de acidente", "Checklist NR-12", "Modelo de laudo técnico", "PPRA/PGR").

4. O documento gerado seguirá o padrão: cabeçalho com nome do documento, nome da SKILL e data; rodapé com **SSTSEMSTSegPrev_CAAO — @oliveiracarlosadv**.
